## üöÄ Overview

This project provides a robust framework for financial analysis. By inputting a few key drivers, the model automatically generates a 5-year projection, balances the Balance Sheet, and calculates the intrinsic value of a business.

## üõ†Ô∏è Key Features

* **Fully Integrated Statements:** Changes flow dynamically across the IS, BS, and CFS.
* **Debt Engine:** Tracks principal repayments and interest expense schedules.
* **Automated DCF:** Built-in calculation for Terminal Value using the Gordon Growth Method.
* **Health Metrics:** Real-time calculation of EBITDA, Working Capital, and Free Cash Flow.

## üíª How to Use

1. **Upload Historicals:** Input your Year 0 data into the statement sheets.
2. **Define Assumptions:** Update the `Inputs` sheet (Revenue Growth, WACC, etc.).
3. **Review Charts:** Check the `Dashboard` and visual charts for automated insights.
